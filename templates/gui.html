Api Gui

This gui has the ability to use all parameters of our Api. If you want to add any companies or topics to the parameters, you can do that by entering a single company or topic in the correlated textbox and clicking the "Add" button next to it. You can then remove any of them by clicking on the "X" button next to the name/topic in the added companies/topics list. You must add dates and can do that by selected a date for both start and end dates and then clicking "Go". Our application will then combine your parameters (adding on "T00:00:00.000Z" to both dates) to create a valid api call and will return all the results (converted from JSON) below (Each article is represented as a headline and link the original article). For this gui you can also add/remove favourites which are stored in cookies.
