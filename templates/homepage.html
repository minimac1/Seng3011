Our aim

Our web application is intended to help users keep up with any significant events that could involve their stocks.


How

Through the use of Google searches as a predictor we can tell whenever an important event happens and combining this with a sentiment analysis of relevant articles during that time period we can accurately determine whether it was a good or bad event.


Our API

We have also developed an api that can be used to retrieve all the information on a particular company within a specified time period. This is used in our application to retrieve a list of related articles so that we can then perform a sentiment analysis.
