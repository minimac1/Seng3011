Parameters
Example url: http://seng3011-turtle.ap-southeast-2.elasticbeanstalk.com/newsapi/v1.0/query?startDate=BLANK&endDate=BLANK&companyId=BLANK&topic=BLANK
Replace each BLANK with an accepted value for the parameter.
Name Description Type Accepted values
startDate The start of the time period for which you would like to search through String A single date in the format: yyyy-MM-ddThh:mm:ss.SSSZ. This field must be completely filled out (the T remains a T).
endDate The end of the time period for which you would like to search through String A single date in the format: yyyy-MM-ddThh:mm:ss.SSSZ. This field must be completely filled out (the T remains a T).
companyID Company/ies of which you would like to search for related news String Either ASX ID e.g. WOW.AX or full name e.g. Woolworths. Note it is not necessary to add the suffix '.AX' to ASX ID's. Replace all spaces with '-' and seperate multiple companies with '_'. This field can be left blank.
topic Topics of which you would like to search for related news. Can also enclose phrases in quotation marks to add articles that include that phrase String Topic names or phrases as strings. Replace all spaces with '-' and seperate multiple topics with '_'. This field can be left blank.
